## v0.0.2

### Features

- **Generics (monomorphization)**: Generic structs (`Box<T>`) and functions (`T identity<T>(T x)`) with full monomorphization at compile time. Supports multiple type parameters and nested generics (`Box<Box<int>>`).
- **Dynamic arrays**: `int[]` type with literal syntax `[1, 2, 3]`, index access/assignment, `push()`, `pop()`, and `len()`.
- **Operators**: `%` (modulo), `&&`/`||` (short-circuit logical), `~` (bitwise NOT).
- **Unreachable code detection**: Warning for statements after `return`, `break`, or `continue`.

### Fixes

- Type checker now reports unknown types with source location.
- Struct literals validate field count and field value types.
- Field assignment validates assigned value type.
- REPL uses `using io` instead of stale `import io`.
- Parser only treats uppercase identifiers as struct literals, fixing `inspect x {}` being misinterpreted.

### Infrastructure

- Golden test framework (`tests/cli/run_golden.sh`) covering arrays, operators, structs, enums, generics, inspect, control flow, and recursion.
- Release workflow now uses `docs/changelog/` for release notes and auto-versions the VSIX from the git tag.
- MIT license added.
