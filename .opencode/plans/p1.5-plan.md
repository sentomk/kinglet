# P1.5 Phase Plan: Type Checker + Typed LiteralExpr

## Overview

P1.5 完成两个核心任务：
1. **Typed LiteralExpr** — 拆分 `LiteralExpr` 为独立节点，消除编译器中的字符串重解析
2. **Type Checker** — 中等范围类型检查（函数签名、返回值、变量类型声明）

## Task 8: Typed LiteralExpr Nodes

### 目标
将 `LiteralExpr`（所有字面量存为 `std::string`）拆分为类型化节点，编译器直接使用已解析的值。

### AST 变更 (`ast.h`)

**删除：**
```cpp
struct LiteralExpr final : Expr {
  std::string value;  // 当前：所有字面量存为字符串
};
```

**新增：**
```cpp
struct IntLiteralExpr final : Expr {
  int64_t value;
};

struct FloatLiteralExpr final : Expr {
  double value;
};

struct StringLiteralExpr final : Expr {
  std::string value;
};

struct BoolLiteralExpr final : Expr {
  bool value;
};

struct NullLiteralExpr final : Expr {
  // 无数据成员
};
```

### Parser 变更 (`parser.cc`)

`primary()` 中根据 Token 类型创建对应节点：

| TokenType | AST 节点 | 值来源 |
|-----------|---------|--------|
| `INTEGER` | `IntLiteralExpr` | `token.int_value` |
| `FLOAT_LIT` | `FloatLiteralExpr` | `token.float_value` |
| `STRING_LIT` | `StringLiteralExpr` | `token_text()` |
| `CHAR_LIT` | `IntLiteralExpr` | `token.int_value` |
| `TRUE` | `BoolLiteralExpr` | `true` |
| `FALSE` | `BoolLiteralExpr` | `false` |
| `NULL_` | `NullLiteralExpr` | — |

### Compiler 变更 (`compiler.cc`)

**删除：**
- `parse_int_literal()` 函数
- `parse_double_literal()` 函数
- `LiteralExpr` 的字符串解析逻辑

**替换为：**
```cpp
if (const auto *int_lit = dynamic_cast<const ast::IntLiteralExpr *>(&expr)) {
  emit_constant(Value::int_value(int_lit->value), int_lit->location);
  return;
}
if (const auto *float_lit = dynamic_cast<const ast::FloatLiteralExpr *>(&expr)) {
  emit_constant(Value::double_value(float_lit->value), float_lit->location);
  return;
}
// ... BoolLiteralExpr, NullLiteralExpr, StringLiteralExpr
```

### 文件清单

| 文件 | 操作 |
|------|------|
| `src/ast/ast.h` | 删除 `LiteralExpr`，新增 5 个类型化节点 |
| `src/ast/ast.cc` | 更新构造函数和 `print()` 实现 |
| `src/parser/parser.cc` | `primary()` 中按 Token 类型创建对应节点 |
| `src/compiler/compiler.cc` | 删除字符串解析函数，用类型化节点替代 |

## Task 7: Type Checker Layer

### 目标
在 Parser 和 Compiler 之间插入类型检查阶段，验证：
- 变量声明的类型与初始化值类型匹配
- 函数参数类型和数量正确
- 函数返回值类型与声明一致
- 算术/比较运算的操作数类型兼容

### 类型表示 (`types.h` 新文件)

```cpp
enum class TypeKind {
  Int,
  Float,
  Bool,
  String,
  Void,
  Null,
  Function,
};

struct Type {
  TypeKind kind;
  // Function 类型额外信息
  std::vector<Type> param_types;
  std::unique_ptr<Type> return_type;
};

// 预定义类型实例
extern const Type INT_TYPE;
extern const Type FLOAT_TYPE;
extern const Type BOOL_TYPE;
extern const Type STRING_TYPE;
extern const Type VOID_TYPE;
extern const Type NULL_TYPE;
```

### TypeChecker 类 (`type_checker.h` 新文件)

```cpp
struct TypeError {
  ast::SourceLocation location;
  std::string message;
};

struct TypeCheckResult {
  std::vector<Type> errors;  // 或 std::vector<TypeError>
  // 可选：为每个 Expr 节点附加推断的类型
};

class TypeChecker {
public:
  TypeCheckResult check(const ast::Program &program);

private:
  void check_function(const ast::FunctionDecl &function);
  void check_stmt(const ast::Stmt &stmt, const Type &expected_return);
  Type check_expr(const ast::Expr &expr);

  void declare_var(const std::string &name, const Type &type, bool is_mutable);
  std::optional<Type> lookup_var(const std::string &name) const;
  void error_at(ast::SourceLocation location, std::string message);

  struct VarInfo {
    Type type;
    bool is_mutable;
  };

  std::vector<std::unordered_map<std::string, VarInfo>> scopes_;
  std::vector<TypeError> errors_;
};
```

### 类型推断规则

| 表达式 | 推断类型 |
|--------|---------|
| `IntLiteralExpr` | `Int` |
| `FloatLiteralExpr` | `Float` |
| `BoolLiteralExpr` | `Bool` |
| `StringLiteralExpr` | `String` |
| `NullLiteralExpr` | `Null` |
| `a + b` (Int, Int) | `Int` |
| `a + b` (Float, Float) | `Float` |
| `a + b` (Int, Float) | `Float`（隐式提升） |
| `a == b` | `Bool` |
| `a < b` | `Bool` |
| `!a` | `Bool` |
| `-a` (Int) | `Int` |
| `-a` (Float) | `Float` |

### 类型检查规则

| 检查项 | 错误信息示例 |
|--------|-------------|
| 变量类型不匹配 | `Cannot assign Float to variable of type Int` |
| 函数参数数量 | `Expected 2 arguments, got 3` |
| 函数参数类型 | `Expected Int, got Float` |
| 返回值类型 | `Cannot return Float from function returning Int` |
| 算术运算类型 | `Cannot add Bool and Int` |
| 未声明变量 | `Undeclared variable 'x'` |
| 重复声明 | `Variable 'x' already declared` |

### 集成到编译流程

**Pipeline 变更：**
```
当前:  Source → Scanner → Parser → Compiler → VM
P1.5:  Source → Scanner → Parser → TypeChecker → Compiler → VM
```

**main.cc 变更：**
```cpp
kinglet::Parser parser(tokens);
kinglet::ParseResult result = parser.parse();
// ... 错误检查 ...

kinglet::TypeChecker checker;
kinglet::TypeCheckResult type_result = checker.check(*result.program);
// ... 类型错误检查 ...

kinglet::Compiler compiler;
kinglet::CompileResult compile_result = compiler.compile(*result.program);
```

## 测试用例

### 正向测试（应通过）
```cpp
int main() {
    int x = 42;
    float y = 3.14;
    bool flag = true;
    int z = x + 10;
    float w = x + y;  // 隐式提升
    return 0;
}
```

### 负向测试（应报错）
```cpp
int main() {
    int x = true;           // Error: Cannot assign Bool to Int
    int y = 3.14;           // Error: Cannot assign Float to Int
    int z = x + true;       // Error: Cannot add Int and Bool
    return 3.14;            // Error: Cannot return Float from Int function
}
```

### 函数测试
```cpp
int add(int a, int b) {
    return a + b;
}

int main() {
    int result = add(1, 2);     // OK
    int bad = add(1, 2, 3);     // Error: Expected 2 arguments, got 3
    int bad2 = add(1, true);    // Error: Expected Int, got Bool
    return result;
}
```
