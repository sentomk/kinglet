# Kinglet Brand Assets

Kinglet uses a rounded technical mark built from:

- a `K` monogram for the language name
- angle brackets for C++-flavored syntax
- stacked bars for bytecode / VM execution
- a small crown notch for the "Kinglet" name

Files:

- `kinglet-mark.svg` — primary app / project icon
- `kinglet-wordmark.svg` — horizontal logo lockup for light backgrounds
- `kinglet-wordmark-light.svg` — horizontal logo lockup for dark backgrounds
- `kinglet-mark-mono.svg` — single-color variant for docs, print, and small UI

Palette:

- Graphite: `#101418`
- Off-white: `#F5F1E8`
- Warm gold: `#F2B84B`
- Electric cyan: `#36D1DC`
